# Dataset

The accelerated-ageing colorimetry dataset (`mogao_data_measured_recovered.xlsx`)
is **not bundled** in this repository. It is distributed as a citable dataset on
figshare:

> Accelerated-ageing colorimetry of Mogao replica pigments — 21-point
> L*a*b* time series over 61 days (vermilion, malachite, azurite) on a 1:5
> lotus pedestal under two environmental arms.
> figshare, DOI: **[to be added on acceptance]**

Download the `.xlsx` file into **this directory** (`experiments/data/`) before
running any script in `experiments/figures/`.
